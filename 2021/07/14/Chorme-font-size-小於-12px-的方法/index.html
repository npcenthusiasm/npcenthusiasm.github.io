Chrome font size 如何小於 12px ？

Posted on 2021-07-14
因為看到設計稿上的字體大小為 10px，所以理所當然直接寫下css font-size 10px，但怎麼設定都沒反應，還發現 font-size 小於 12px 就不會變了...

因為看到設計稿上的字體大小為 10px，所以理所當然直接寫下css font-size: 10px，但怎麼設定都沒反應，還發現 font-size 小於 12px 就不會變了。後來 google 了才發現是 chrome 在搞鬼，查到文章表示 chrome 認為字體低於 12px 對人類在閱讀上是不友善的，所以加上了最小字體的限制，所以只要解開限制就好了，但真的是這樣嗎？

第一種解法 (不推薦)

在 chorme 網址列輸入 chrome://settings/fonts，就會看到以下畫面

可以看到預設最小字體被限制為 12px，所以把最小字體的限制調小一點就可以了

缺點是：調完後只有自己看得到，若造訪你網頁的 user 沒更改上方的最小字體限制的話，是不會有任何效果的，所以這個解法是治標不治本的作法，但你可以幫業主調，用來應付業主。

第二種解法

另外一種方法是用 css 的 transform: scale() 方法，只要算出縮小的比例就好了，不過最好多測試一下各個瀏覽器有沒有問題。

先設定 font-size: 12px 之後，接下來就用 12px 去算縮小的比率

  • 例如 10px， 10/12 = 0.833 這樣
  • 例如 9px， 9/12 = 0.75 這樣

因為是用 scale 去縮小，所以原本用 margin、padding 調的間距也會跑掉，所以我用 transform-origin: left top;，讓這區塊以左上角為中心縮放，讓畫面上對齊好。

結論

這個情況通常是遇到才會知道，建議可以先跟設計師討論字體是否真的有需要設定到這麼小，也許討論完後就不用解這問題了，但如果真的需要的話，那之後也知道怎麼解了。
