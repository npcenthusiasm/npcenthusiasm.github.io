用 hexo 建立一個 blog app

Posted on 2021-05-21

在建立完之後，想要紀錄一下自己建立起來的過程，剛好可以當第一篇文章哈哈

我們將透過 hexo-cli 建立整個 blog app

筆者 node -v 為 v14.15.1

  1. 首先安裝 hexo-cli，-g 將會全域安裝

  2. init hexo app
    先 cd 到你想要建立的資料夾目錄底下，

    安裝完後的畫面會長這樣

  3. 預覽剛剛建立好的 app

建立一篇文章

會在 source/_posts/ 底下建立一個檔名為標題的 md 檔

撰寫文章的方式為 markdown 的格式

更改主題

先安裝主題！

安裝完主題之後，接下要將預設主體改成剛剛下載的新主題

將 _config.yml 中的 theme: landscape 改成 theme: next

改完後重新 run 一次 hexo server，就會看到主題已經更新了

發佈到 github 上

先在自己的 github 上建立一個 repository
名為 你的github帳號.github.io

先安裝一個快速發佈到 github 上的套件

安裝完後去 _config.yml 設定要推到 github 上的 repo url

設定完後就可以推上去了
